# Meeting Assistant Monorepo

## Stack
- Monorepo with `pnpm` workspaces
- Frontend: Next.js (`apps/frontend`)
- Backend: NestJS (`apps/backend`)
- ORM: Prisma
- DB: PostgreSQL
- Auth: JWT access token (15m) + refresh token (7d, httpOnly cookie)
- Multi-tenant: Organization + Membership
- Social login ready: Google OAuth strategy
- Deployment: Render (`render.yaml`)

## Monorepo Structure

```text
/apps
  /backend
  /frontend
/packages
  /shared
```

## Backend Overview

- `apps/backend/src/main.ts`
  - Helmet
  - CORS
  - Rate limiting
  - Global `ValidationPipe`
  - Global error filter
  - Logging middleware

- `apps/backend/src/modules/auth`
  - `AuthController`
  - `AuthService`
  - `JwtStrategy`
  - `RefreshTokenStrategy`
  - `GoogleStrategy`

- `apps/backend/src/modules/organizations`
  - organization creation/listing

- `apps/backend/src/modules/memberships`
  - membership listing with organization scope

- `apps/backend/src/modules/users`
  - current user endpoint

- `apps/backend/src/modules/meeting`
  - preserves MVP audio processing endpoint: `POST /api/process-audio`

- `apps/backend/src/common/guards`
  - `JwtAuthGuard` global
  - `OrganizationGuard` global

- `apps/backend/src/common/decorators`
  - `@Public()`
  - `@SkipOrganization()`
  - `@CurrentUser()`
  - `@CurrentOrganization()`

## Prisma

Models:
- `User`

- `Session`

Notes:
- Keeps migration safe-first approach.

## Frontend Overview

- `apps/frontend/lib/api.ts`
  - Axios baseURL from `NEXT_PUBLIC_API_URL`
  - Authorization header injection
  - silent refresh retry on 401

- `apps/frontend/lib/auth.ts`
  - in-memory access token handling
  - refresh call with cookie (`/auth/refresh`)

- `apps/frontend/middleware.ts`
  - route protection scaffold

## Environment

Use root `.env.example` as base.

Required keys:
- `DATABASE_URL`
- `JWT_SECRET`
- `REFRESH_SECRET`
- `GOOGLE_CLIENT_ID`
- `GOOGLE_CLIENT_SECRET`
- `NEXT_PUBLIC_API_URL`

## Local Run

1. Install dependencies:
```bash
pnpm install
```

2. Generate Prisma client:
```bash
pnpm prisma:generate
```

3. Build all:
```bash
pnpm build
```

4. Start backend:
```bash
pnpm dev:backend
```

5. Start frontend:
```bash
pnpm dev:frontend
```

## Render

`render.yaml` provisions:
